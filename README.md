🍡 Games Included
1. 🍭 Candy Match (candy-match.html)
A match-3 puzzle game in the Candy Crush tradition. Swap adjacent candies to line up three or more of the same kind — they pop, new candies fall, and chain reactions stack up the score.

Features

8×8 grid with 6 candy types (🍬🍭🍫🍩🍪🧁)
Cascading combos with score multipliers (x2, x3, x4…)
Combo messages: DOUBLE, TRIPLE, MEGA, ULTRA, CRAZY
💡 Hint button — highlights a valid move when you're stuck
🏆 Personal best saved in localStorage
End-game badges: Sugar Apprentice, Candy Master, Sweet Legend, Combo King
Smooth animations: wiggle-on-select, pop-on-match, bounce-on-fill
How to Play

1.
Click any candy to select it (it wiggles and glows yellow).
2.
Click an adjacent candy (up/down/left/right) to swap.
3.
If 3+ same candies line up → they pop and you score.
4.
Invalid swaps auto-revert and refund your move.
5.
New candies fall from the top — chain reactions stack multipliers!
6.
You have 30 moves — get the highest score possible.
